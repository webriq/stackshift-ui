# @stackshift-ui/popover

Displays rich content in a portal, triggered by a button.

Please refer to the [documentation](https://stackshift-ui.webriq.com/docs/components/popover) for more information.

## Installation

```sh
yarn add @stackshift-ui/popover
# or
npm i @stackshift-ui/popover
```
