# @stackshift-ui/pagination

Pagination with page navigation, next and previous links.

Please refer to the [documentation](https://stackshift-ui.webriq.com/docs/components/pagination) for more information.

## Installation

```sh
yarn add @stackshift-ui/pagination
# or
npm i @stackshift-ui/pagination
```
