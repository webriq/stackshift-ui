# @stackshift-ui/sheet

Extends the Dialog component to display content that complements the main content of the screen.

Please refer to the [documentation](https://stackshift-ui.webriq.com/docs/components/sheet) for more information.

## Installation

```sh
yarn add @stackshift-ui/sheet
# or
npm i @stackshift-ui/sheet
```
