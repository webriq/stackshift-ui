# @stackshift-ui/date-picker

A date picker component with range and presets.

Please refer to the [documentation](https://stackshift-ui.webriq.com/docs/components/date-picker) for more information.

## Installation

```sh
yarn add @stackshift-ui/date-picker
# or
npm i @stackshift-ui/date-picker
```
