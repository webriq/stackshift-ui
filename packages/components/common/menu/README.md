# @stackshift-ui/menu

A collection of links for navigating websites.

Please refer to the [documentation](https://stackshift-ui.webriq.com/docs/components/menu) for more information.

## Installation

```sh
yarn add @stackshift-ui/menu
# or
npm i @stackshift-ui/menu
```
