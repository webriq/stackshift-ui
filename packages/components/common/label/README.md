# @stackshift-ui/label

Renders an accessible label associated with controls.

Please refer to the [documentation](https://stackshift-ui.webriq.com/docs/components/label) for more information.

## Installation

```sh
yarn add @stackshift-ui/label
# or
npm i @stackshift-ui/label
```
