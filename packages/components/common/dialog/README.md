# @stackshift-ui/dialog

A window overlaid on either the primary window or another dialog window, rendering the content underneath inert.

Please refer to the [documentation](https://stackshift-ui.webriq.com/docs/components/dialog) for more information.

## Installation

```sh
yarn add @stackshift-ui/dialog
# or
npm i @stackshift-ui/dialog
```
