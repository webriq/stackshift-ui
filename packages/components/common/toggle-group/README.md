# @stackshift-ui/toggle-group

A set of two-state buttons that can be toggled on or off.

Please refer to the [documentation](https://stackshift-ui.webriq.com/docs/components/toggle-group) for more information.

## Installation

```sh
yarn add @stackshift-ui/toggle-group
# or
npm i @stackshift-ui/toggle-group
```
