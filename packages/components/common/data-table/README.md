# @stackshift-ui/data-table

Powerful table and datagrids built using TanStack Table.

Please refer to the [documentation](https://stackshift-ui.webriq.com/docs/components/data-table) for more information.

## Installation

```sh
yarn add @stackshift-ui/data-table
# or
npm i @stackshift-ui/data-table
```
