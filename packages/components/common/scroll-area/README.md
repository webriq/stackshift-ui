# @stackshift-ui/scroll-area

Augments native scroll functionality for custom, cross-browser styling.

Please refer to the [documentation](https://stackshift-ui.webriq.com/docs/components/scroll-area) for more information.

## Installation

```sh
yarn add @stackshift-ui/scroll-area
# or
npm i @stackshift-ui/scroll-area
```
