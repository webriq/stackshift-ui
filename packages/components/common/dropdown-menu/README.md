# @stackshift-ui/dropdown-menu

Displays a menu to the user — such as a set of actions or functions — triggered by a button.

Please refer to the [documentation](https://stackshift-ui.webriq.com/docs/components/dropdown-menu) for more information.

## Installation

```sh
yarn add @stackshift-ui/dropdown-menu
# or
npm i @stackshift-ui/dropdown-menu
```
