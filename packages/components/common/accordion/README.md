# @stackshift-ui/accordion

A vertically stacked set of interactive headings that each reveal a section of content.

Please refer to the [documentation](https://stackshift-ui.webriq.com/docs/components/accordion) for more information.

## Installation

```sh
yarn add @stackshift-ui/accordion
# or
npm i @stackshift-ui/accordion
```
