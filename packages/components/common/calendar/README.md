# @stackshift-ui/calendar

A date field component that allows users to enter and edit date.

Please refer to the [documentation](https://stackshift-ui.webriq.com/docs/components/calendar) for more information.

## Installation

```sh
yarn add @stackshift-ui/calendar
# or
npm i @stackshift-ui/calendar
```
