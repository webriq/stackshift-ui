# @stackshift-ui/toast

An opinionated toast component for React.

Please refer to the [documentation](https://stackshift-ui.webriq.com/docs/components/toast) for more information.

## Installation

```sh
yarn add @stackshift-ui/toast
# or
npm i @stackshift-ui/toast
```
