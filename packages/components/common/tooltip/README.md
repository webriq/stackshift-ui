# @stackshift-ui/tooltip

A popup that displays information related to an element when the element receives keyboard focus or the mouse hovers over it.

Please refer to the [documentation](https://stackshift-ui.webriq.com/docs/components/tooltip) for more information.

## Installation

```sh
yarn add @stackshift-ui/tooltip
# or
npm i @stackshift-ui/tooltip
```
